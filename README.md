Niigata.NET 2015-10 サンプル
=====

[Niigata.NET 2015-10 - Niigata.NET | Doorkeeper](https://ngtnet.doorkeeper.jp/events/30553)で行ったセッション「業務アプリケーション開発を支える.NET技術」で使用したデモコードです。

- `MEFSample`  
  MEF(Managed Extensibility Framework)のサンプル
- `WCFSample`  
  WCF(Windows Communication Foundation)のサンプル

ビルドや実行に関する注意点は、それぞれのサンプル内の`README.md`を参照してください。
